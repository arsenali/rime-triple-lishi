# rime-triple-lishi
李氏三拼方案，三码输入一个带调音节

有26键和15键两个版本，前者主要用于电脑，后者主要用于手机。

两个版本都用三码输入一个带调音节，26键版为“声+韵+调”方案，第三码声调可省略；15键版为“声+介调+韵”方案，声调不可省略。26键用于qwerty键盘，15键可用于数字键盘和qwerty键盘，左手、右手、双手操作均可。

详见阿森纳里的简书“李氏三拼”专题。
